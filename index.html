2026

Research

Brontes

Brontes is a waveform-domain enhancement model that repairs neural codec artifacts while upsampling degraded 24 kHz speech into 48 kHz target audio. It matters because it replaces artifact-preserving enhancement behavior with synthesis-first restoration aimed at perceptual realism.

  • Synthesis-first stack with selective deep skips, temporal compression, and LSTM bottleneck
  • Gated decoder fusion with adversarial objectives in time and spectral domains
  • Stabilized training through discriminator representation changes
Code Technical Report

2026

Research

VITS EVOlution (VITS EVO)

VITS EVOlution is my research-driven optimization of the VITS TTS architecture, built to deliver extreme real-time speed while keeping audio quality practical for real products. It reaches about 300x real-time synthesis on an NVIDIA A6000 and demonstrates a deployment-ready path through ONNX + DirectML in TensorVox.

  • Architecture and inference optimizations targeted at low-latency, real-time use
  • Model is private, but production integration is documented in TensorVox

A text-to-speech model trained on concatenated text-audio tokens in a decoder-only LLM-style setup. It matters because it pushes efficient, high-fidelity audio language modeling toward practical throughput.

MQGAN / MusicLSTM

Experimental audio tokenization around a spectrogram codec with a ResNet compressor and U-Net refiner. It matters because reconstruction quality is improved without abandoning compact tokenized workflows.

  • Spectrogram-first codec framing for music-oriented generation tests
  • Compressor-refiner split for better detail recovery in outputs
